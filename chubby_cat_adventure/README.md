# 🐱 胖妹大闯关

一款双人合作闯关的 Canvas 2D 猫咪冒险游戏，支持单人、本地双人和在线联机模式。

## 🎮 游戏简介

两只勇敢的猫咪——**小胖**（狸花猫）和**妹**（布偶猫）携手闯过 10 个关卡，击败各路 Boss，拯救猫咪世界！

## 🕹️ 游戏模式

| 模式 | 说明 |
|------|------|
| 🎯 单人冒险 | 一人操控一只猫，独自闯关 |
| 👥 本地双人 | 同一设备，两人分别操控两只猫合作闯关 |
| ⚔️ 玩家决斗 | 本地双人对战 PVP，支持 1/3/5 局制 |
| 🌐 在线联机 | 通过 PeerJS (WebRTC) 点对点联机，分享房间码即可加入 |

## 🐱 角色介绍

### 小胖（狸花猫）
- 性格活泼，身手敏捷
- **远程**：吐毛球
- **近战**：挥爪子（1 伤害）
- **大招**：冲击波（2 伤害，击退周围怪物）
- **被动**：受伤时自动释放冲击波，震开周围怪物

### 妹（布偶猫）
- 优雅灵动，擅长隐匿
- **远程**：吐毛球
- **近战**：挥爪子
- **大招**：隐身（5 秒无敌，移动速度翻倍）
- **被动**：受伤时自动隐身 5 秒，移动速度翻倍

## 🎹 操作方式

### 键盘控制

| 操作 | 小胖 | 妹 |
|------|------|-----|
| 移动 | W A S D | ↑ ↓ ← → |
| 远程攻击 | F | L |
| 近战攻击 | G | K |
| 大招 | Q | U |
| 救助队友 | R | O |
| 暂停 | ESC | ESC |

### 其他输入
- 🎮 支持手柄操控（自动检测）
- 📱 支持手机触屏虚拟摇杆和按钮

## 👾 怪物与 Boss

- **史莱姆**：基础怪物，数量多
- **骷髅**：攻击力较强的精英怪
- **Will**：毒液女王，擅长吐毒液形成持续伤害区域
- **Loading**：光波巨人，发射震荡光波震晕玩家
- **狂暴版 Boss**：头发炸起，周身能量光环，攻击大幅强化
- **觉醒版 Boss**（隐藏关）：全新 AI，可被击败后加入己方阵营

## 🎁 道具系统

击杀怪物有概率掉落道具：

| 道具 | 效果 |
|------|------|
| 👟 加速鞋 | 移动速度提升 |
| ❤️ 爱心 | 恢复血量 |
| ⏳ 沙漏 | 重置大招冷却时间 |

## 🏆 成就系统

涵盖多个维度：
- **击杀成就**：初试锋芒、杀手系列
- **连击成就**：3/5/10/15 连击
- **关卡成就**：通关第 1/5/10 关
- **完美成就**：无伤通关
- **Boss 成就**：击败全部 Boss
- **救援成就**：救助队友
- **道具成就**：收集道具

成就数据保存在浏览器 localStorage 中，通关隐藏关可解锁特殊成就。

## 🌟 游戏特色

- 纯前端实现，单 HTML 文件，无需服务器
- Canvas 2D 渲染，支持高分辨率屏幕
- WebRTC (PeerJS) 在线联机，无需注册
- 手柄支持，自动检测
- 移动端适配，虚拟摇杆 + 按键
- 新手引导教程
- 全屏模式支持
- 音效系统（Web Audio API）
- 角色图鉴 & 成就系统

## 🚀 在线游玩

将 `chubby_cat_adventure.html` 部署到任意静态托管服务即可通过链接访问：

1. 上传文件到 GitHub 仓库
2. 在仓库 Settings → Pages 中启用 GitHub Pages
3. 访问 `https://<username>.github.io/<repo>/chubby_cat_adventure.html`

## 💻 本地运行

无需安装任何依赖，直接用浏览器打开或启动本地服务器：

```bash
# 方式一：直接用浏览器打开
# 双击 chubby_cat_adventure.html 即可

# 方式二：本地服务器（推荐，联机功能需要）
cd game
python -m http.server 8080
# 然后访问 http://localhost:8080/chubby_cat_adventure.html
```

> ⚠️ 联机功能需要通过 HTTP/HTTPS 协议访问，直接打开文件（file://）时联机可能不可用。
